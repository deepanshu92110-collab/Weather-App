# 🌦️ Weather Dashboard Web App

## 📌 Project Overview

The **Weather Dashboard Web App** is a responsive web application that allows users to check real-time weather conditions for any city around the world. Users can search for a location and get detailed weather information such as temperature, humidity, wind speed, and overall weather conditions.

This project focuses on building a clean UI, integrating APIs, and implementing features like search, filtering, and dynamic data rendering.

---

## 🎯 Purpose

The purpose of this project is to:

* Practice working with real-world APIs
* Build a responsive and interactive web application
* Implement features like search, filtering, and sorting
* Strengthen frontend development skills using modern tools

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Display current temperature and weather condition
* 💨 Show wind speed and humidity
* 📅 Display date and time of location
* 🌍 Support for multiple cities worldwide
* 📱 Responsive design (mobile + desktop)
* ⭐ (Future Scope)

  * 7-day weather forecast
  * Favorite cities list
  * Dark/light mode toggle
  * Sorting cities by temperature or weather condition
